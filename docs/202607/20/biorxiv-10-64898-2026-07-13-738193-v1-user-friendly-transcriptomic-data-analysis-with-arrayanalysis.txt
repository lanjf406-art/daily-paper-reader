# ABSTRACT 22 

Transcriptomic profiling has become a cornerstone of modern biomedical research. To make 23 

transcriptomic analyses accessible to a broader scientific community, specifically including 24 

researchers with limited bioinformatics expertise, we introduced ArrayAnalysis in 2013 as a 25 

user -friendly web -based application for microarray data analysis. We now present a major 26 

update ( https://arrayanalysis.org ), introduc ing a strongly interactive platform that facilitates 27 

the dedicated exploration and analysis of both microarray and RNA -seq data, and allows for 28 

the generation of publication -ready outputs. Users can perform key analysis steps, including 29 

data pre -processing and quality control, differential expression analysis, and gene set analysis, 30 

via a sequential, interactive workflow . At each step, the application provides interactive 31 

visualizations accompanied by information pages to support interpretation. Users can 32 

dynamically adjust figure layouts and colour palettes and export figures as vector graphics and 33 

high -resolution raster images. For non -expert users, ArrayAnalysis offers step -by -step 34 

guidance to support correct usage and facilitate learning, while for experienced 35 

bioinformaticians, it provides a streamlined and flexible workflow ideal for large -scale analyses 36 

requiring efficient and consistent processing. ArrayAnalysis is available both as a web 37 

application and for local deployment as a desktop application, Docker image, or R package, 38 

making it suitable for diverse computational environments, user groups, and analytical 39 

purposes. Together, ArrayAnalysis empowers a broad community of biomedical researchers to 40 

unlock the full potential of transcriptomic data. 41

# BACKGROUND 45 

Gene expression profiling, through microarray or RNA sequencing (RNA -seq ) technologies, 46 

has become an indispensable tool in biomedical research for uncovering molecular 47 

mechanisms and identifying biomarkers. However, the analysis of transcriptomic data remains 48 

challenging for many researchers, especially for those without formal training in bioinformatics. 49 

To bridge this gap, several applications have been developed that offer a user -friendly 50 

interface to the analysis of transcriptomic data. As one of these initiatives, we introduced 51 

ArrayAnalysis , in 2013 as a web -based application designed as a user -friendly solution for 52 

microarray data analysis [1] . While this earlier version helped many researchers analyse 53 

microarray experiments more easily, it did not support RNA -seq data and lacked more 54 

advanced interactive features. More recently, other applications have been developed for the 55 

analysis of RNA -seq data [2 -5] .56 

57 

Although these applications lower the barrier to entry, they still have important limitations. 58 

Many provide little guidance to help users navigate the analysis workflow, making them less 59 

suitable for researchers inexperienced with gene expression profiling. In addition, the output 60 

of such tools is often rigid, with limited options to support interactive data exploration and to 61 

customi se figures for publication -quality presentation. To address these limitations, we 62 

developed a new version of ArrayAnalysis , a web -based application that provides a63 

streamlined and accessible workflow for the analysis of both microarray and RNA -seq data. 64 

ArrayAnalysis offers several innovations, including enhanced user guidance throughout the 65 

analysis steps, interactive outputs that facilitate data exploration, and customi sable figures 66 

tailored for use in scientific publications. With these features, ArrayAnalysis aims to make 67

# IMPLEMENTATION 71 

We developed ArrayAnalysis ( https://arrayanalysis.org ) as a tool for user -friendly analysis of 72 

transcriptomic data. The platform is built with the Shiny framework in the R programming 73 

language ( http://www.r -project.org ) and integrates more than 30 packages from CRAN [6 -28] 74 

and Bioconductor [29 -43] (Table 1 ). ArrayAnalysis is made available as a web application and 75 

for local installation as a desktop app, Docker image, and R package. 76 

77 

Input 78 

In the first step of the ArrayAnalysis app, users can upload expression data and associated 79 

metadata. Microarray data can be provided as a zipped folder containing CEL files, as an 80 

expression matrix (.tsv/.csv/.txt), or as a GEO series matrix file. RNA -seq data can be uploaded 81 

as a matrix of raw or normalized counts (.tsv/.csv/.txt). The metadata, which contains 82 

information about the samples such as the experimental group and age, can be provided as a 83 

table (.tsv/.csv/.txt) or as a GEO series matrix file. Upon uploading the data, a preview of both 84 

the expression data and metadata is displayed, allowing users to verify that the files have been 85 

correctly loaded. If necessary, the metadata can be modified within the application. 86 

87 

ArrayAnalysis includes two demo datasets from the Gene Expression Omnibus (GEO) [44] that 88 

can be directly loaded within the app: GSE6955 [45] (microarray data) and GSE128380 [46] 89 

(RNA -seq data). Both datasets comprise gene expression profil es of post -mortem brain tissue 90 

from Rett syndrome patients and age -matched controls. The GSE6955 and GSE128380 91

# RESULTS 114 

We propose ArrayAnalysis as a user -friendly solution for transcriptomic data analysis. 115 

Following data upload, users can analyse their data in three sequential steps: 1) pre -processing 116 

and quality control, 2) differential gene expression analysis, and 3) gene set analysis. In the 117 

following sections, we provide a detailed description of these three functionalities and perform 118 

a comparison with other existing transcriptomics data analysis tools. 119

Pre -processing and quality control 120 

In the pre -processing and quality control module, the user can prepare the data for 121 

downstream analysis, including interactive filtering of genes and samples as well as 122 

normalization and transformation of expression data. In addition, the data are automatically 123 

screened for potential outlying samples. If potential outliers are detected, a warning message 124 

is displayed to alert the user. An outlier warning is provided when there is at least one sample 125 

that deviates by more than six median absolute deviations from the median on at least one 126 

principal component that explains ≥10% of the total variance. 127 

128 

Several quality control outputs are generated to assist users in evaluating the data quality and 129 

identifying potential outliers. These include boxplots of gene expression distributions, density 130 

plots, sample –sample correlation heatmaps, and principal component analysis (PCA) plots 131 

(Figure 2 ). Furthermore, an interactive gene expression matrix is provided that allows users to 132 

query specific genes and visualize their expression patterns across experimental groups. For 133 

analyses starting from a raw RNA -seq count matrix, an additional bar chart depicting total read 134 

counts per sample is included to help identify discrepancies in sequencing depth. 135 

136 

Differential gene expression analysis 137 

In the differential gene expression analysis module, users can select the statistical 138 

comparison(s) of interest, select covariates, and map between gene identifier types . The 139 

DESeq2 R/Bioconductor package [31] is used for the differential gene expression analysis of 140 

raw RNA -seq counts. For other input data, the limma R/Bioconductor package [33] is applied. 141 

The gene identifier mappings are extracted from the Ensembl database using the biomaRt 142 

R/Bioconductor package [40] . If the connection to the Ensembl database fails (e.g., due to 143

Gene set analysis 163 

Following the differential gene expression analysis, ArrayAnalysis uses the clusterProfiler 164 

R/Bioconductor package [39] to support two types of gene set analysis: overrepresentation 165 

analysis and gene set enrichment analysis. These analyses can be performed on gene 166 

ontologies (GO -BP, GO -MF, and GO -CC) [47] , KEGG [48] , and WikiPathways [49] gene sets. In 167 

this module, an interactive statistics table, bar chart, and gene set similarity network are 168 

provided ( Figure 4 ). 169

# DISCUSSION 185 

We developed a new version of ArrayAnalysis ( https://arrayanalysis.org ) as a user -friendly 186 

solution for transcriptomic data analysis. Compared to the first release in 2013 [1] , the current 187 

version has been substantially updated ; besides offering support for RNA -seq data, it 188 

introduces a strongly interactive analysis workflow that facilitates efficient data exploration 189 

and analysis. It further addresses key limitations of existing tools by providing extensive user -190 

guidance and customi sable, publication -ready outputs. Finally, in addition to the web 191 

application, ArrayAnalysis can be r un locally as an R package, desktop app lication , and Docker 192 

image, enabling secure in -house processing of sensitive data. 193 

194 

A central aim of ArrayAnalysis is to support a broad community of users, ranging from 195 

biomedical researchers without prior bioinformatics training to experienced data analysts. For 196 

users with less bioinformatics experience , the platform offers elaborate guidance features, 197 

including in -app information boxes, tutorials, FAQs, and extensive documentation. In addition, 198 

automated outlier detection helps users recognize potential issues in their data. 199 

200 

For expert users, ArrayAnalysis provides a streamlined and flexible workflow. Options such as 201 

sample exclusion, metadata editing, and figure customi sation can be applied directly within 202 

the application, reducing the need to alternate between multiple tools. The strength of 203 

ArrayAnalysis lies in providing a streamlined and robust workflow for core transcriptomic data 204 

analysis functions. Moreover, to support the use of more specialized tools for advanced 205 

downstream tasks, results can be downloaded at each step and are designed to ensure 206 

interoperability with other applications. For example, differential expression statistics are 207 

provided in a consistent tabular format across different input data types and include standard 208

requirements. The option to export session information and user -selected analysis settings 220 

ensures transparency and reproducibility, as well as aiding manuscript preparation. 221

# CONCLUSIONS 223 

In conclusion, ArrayAnalysis provides both wet -lab scientists and bioinformaticians with a 224 

powerful, intuitive, and freely available tool for transcriptomic data analysis. By lowering 225 

barriers to entry while also supporting expert -level workflows, ArrayAnalysis contributes to 226 

more widespread, reproducible, and effective use of transcriptomics in biomedical research. 227
